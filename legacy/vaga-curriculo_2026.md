# Vaga — Software Engineer (Full Stack)

## Empresa

Empresa de tecnologia focada em desenvolvimento de produtos digitais e plataformas SaaS para diferentes segmentos de mercado.

## Modelo de trabalho

Remoto

## Descrição da vaga

Estamos buscando um Software Engineer Full Stack para atuar no desenvolvimento de aplicações modernas baseadas em tecnologias web e cloud.

O profissional atuará na construção de produtos digitais completos, participando desde a definição da arquitetura até a implementação e deploy das aplicações.

Buscamos alguém com perfil de engenheiro de produto, capaz de integrar frontend, backend, banco de dados e serviços externos para resolver problemas reais de usuários.

## Responsabilidades

- Desenvolver aplicações web modernas utilizando React ou frameworks baseados em React
- Construir APIs REST e integrar sistemas externos
- Trabalhar com bancos de dados SQL
- Participar da arquitetura e evolução de produtos SaaS
- Desenvolver aplicações escaláveis e de alta performance
- Colaborar na integração de funcionalidades baseadas em inteligência artificial e APIs externas
- Participar do ciclo completo de desenvolvimento de software

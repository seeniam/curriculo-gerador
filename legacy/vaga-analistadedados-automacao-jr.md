Com 15 anos de história, nascemos em São Bernardo do Campo e hoje atuamos também em Brasília. Transicionamos do offline para o digital e nos especializamos em dados para comunicação. Somos uma agência de marketing político-eleitoral focada no campo democrático e progressista. Nossa missão é impulsionar campanhas com impacto positivo e elevar vozes que compartilham nossos valores.

Resumo da Vaga

Buscamos um profissional híbrido entre análise de dados e automação. O foco não é o desenvolvimento de sistemas complexos, mas resolver problemas práticos usando ferramentas de IA, scripts rápidos e visualização de dados.

Principais Responsabilidades

Inteligência de Dados: Criar, manter e atualizar dashboards no Power BI.
Operação de IA: Configurar e monitorar Agentes de IA (GPTs, Claude Code, Antigravity, etc.) para otimizar fluxos internos.
Desenvolvimento Web Simples: Criar e ajustar Landing Pages (HTML/CSS básico e/ou React + TailwindCSS).
Automações: Escrever e rodar scripts pontuais em Python para tratamento de dados ou integração de APIs.
Requisitos Técnicos

Power BI (Avançado): Domínio de DAX e modelagem de dados.
Python (Básico/Intermediário): Scripts para automação de tarefas simples.
IA Generativa: Engenharia de prompt e operação de ferramentas de agentes autônomos.
Web Design Básico: Noções de HTML, CSS e funcionamento de Landing Pages.
Diferenciais

SQL (Básico/Intermediário): Consultas em bancos de dados para extração de informações.
APIs de LLMs: Conexão de scripts Python às APIs da OpenAI, Anthropic ou Google (Gemini).
Curadoria de Conteúdo de IA: Estar atualizado com novas bibliotecas e modelos semanais.
Limpeza de Dados (Pandas/NumPy): Experiência em limpar planilhas "sujas" antes de automatizar.

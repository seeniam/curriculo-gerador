# Vaga de Exemplo: Senior DevSecOps Engineer

**Company:** Cybersecurity Innovators Inc.
**Location:** Remote

## Overview
We are looking for a Senior DevSecOps Engineer to lead our security initiatives across hybrid cloud environments. You will be responsible for integrating security into our CI/CD pipelines and managing our cloud infrastructure security posture.

## Responsibilities
- Architect and implement secure cloud infrastructure on AWS (IAM, VPC, EC2, GuardDuty, Macie).
- Lead the implementation of DevSecOps practices within CI/CD pipelines (Jenkins, GitLab CI).
- Perform proactive threat hunting and incident response.
- Manage SIEM solutions (Splunk, Elasticsearch) for log monitoring and analysis.
- Ensure compliance with ISO 27001, SOC 2, and data protection regulations (GDPR).
- Automate security tasks using Python and Bash.

## Requirements
- 8+ years of experience in IT infrastructure and at least 4 years focused on Cybersecurity/DevSecOps.
- Deep knowledge of AWS cloud security architecture.
- Hands-on experience with SIEM, EDR, and Vulnerability Management tools.
- Strong scripting skills in Python for security automation.
- Familiarity with securing containerized environments (Docker, Kubernetes).
- Certifications such as CISSP, AWS Certified Security, or equivalent are highly desired.

Automação de migração de dados com OCR

Problema

Sistema SAE não permitia exportação direta de dados.

Dados de alunos estavam disponíveis apenas em interface visual.

Mais de 5000 registros precisavam ser migrados.

Processo manual

300 prints

OCR manual

união manual de planilhas

Tempo médio

aproximadamente 3 dias de trabalho.

Solução

Desenvolvimento de automação em Python utilizando

PyAutoGUI

OpenCV

Tesseract OCR

openpyxl

Fluxo automatizado

captura de tela -> OCR -> geração de planilha -> consolidação automática

Resultado

redução do processo de aproximadamente 3 dias para cerca de 1h30
